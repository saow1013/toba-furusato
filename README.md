## GitHub Pages デプロイ手順

1. GitHubに新規リポジトリを作成（例：`toba-furusato`）
2. 上記ファイルをリポジトリにプッシュ
3. リポジトリの **Settings → Pages → Source** を `main` ブランチに設定
4. 自動付与されるURL（例：`https://username.github.io/toba-furusato/`）で公開確認
